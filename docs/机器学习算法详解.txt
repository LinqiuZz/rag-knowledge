# 机器学习算法详解

## 一、监督学习算法

### 1.1 线性回归 (Linear Regression)

线性回归是最基础的回归算法，用于预测连续数值。

**数学模型：**
y = w₁x₁ + w₂x₂ + ... + wₙxₙ + b

**损失函数：** 均方误差 (MSE)
L = (1/n) Σ(yᵢ - ŷᵢ)²

**优化方法：**
- 梯度下降法
- 正规方程法

**正则化变体：**
- Lasso回归 (L1正则化)：可以产生稀疏解
- Ridge回归 (L2正则化)：防止过拟合
- Elastic Net：结合L1和L2

### 1.2 逻辑回归 (Logistic Regression)

虽然名字中有"回归"，但逻辑回归是分类算法。

**核心公式：**
P(y=1|x) = σ(wᵀx + b) = 1 / (1 + e^(-wᵀx - b))

**特点：**
- 输出具有概率意义
- 可以使用交叉熵损失函数
- 适合二分类问题

### 1.3 决策树 (Decision Tree)

决策树是一种树形结构的分类和回归算法。

**分裂准则：**
- 信息增益 (ID3)
- 信息增益比 (C4.5)
- 基尼系数 (CART)

**剪枝方法：**
- 预剪枝：提前停止生长
- 后剪枝：先生长再修剪

### 1.4 随机森林 (Random Forest)

随机森林是集成学习方法，基于Bagging思想。

**核心思想：**
1. 从原始数据中Bootstrap抽样
2. 对每个样本训练一棵决策树
3. 每次分裂时随机选择特征子集
4. 最终结果取平均（回归）或投票（分类）

**优点：**
- 准确率高
- 不容易过拟合
- 可以处理高维数据
- 可以评估特征重要性

### 1.5 支持向量机 (SVM)

SVM寻找最优超平面来分隔不同类别的数据。

**核心概念：**
- 支持向量：距离超平面最近的数据点
- 间隔：支持向量到超平面的距离
- 核函数：将数据映射到高维空间

**常见核函数：**
- 线性核：K(x,y) = xᵀy
- 多项式核：K(x,y) = (xᵀy + c)ᵈ
- RBF核：K(x,y) = exp(-γ||x-y||²)

### 1.6 K近邻 (KNN)

KNN是一种基于实例的学习算法。

**算法步骤：**
1. 计算待分类样本与所有训练样本的距离
2. 选择距离最近的K个样本
3. 统计K个样本中各类别的数量
4. 返回数量最多的类别

**距离度量：**
- 欧氏距离：d = √(Σ(xᵢ-yᵢ)²)
- 曼哈顿距离：d = Σ|xᵢ-yᵢ|
- 闵可夫斯基距离：d = (Σ|xᵢ-yᵢ|ᵖ)^(1/p)

## 二、无监督学习算法

### 2.1 K-means聚类

K-means是最常用的聚类算法。

**算法步骤：**
1. 随机选择K个初始中心点
2. 将每个数据点分配到最近的中心点
3. 重新计算每个簇的中心点
4. 重复步骤2-3直到收敛

**评估指标：**
- 轮廓系数 (Silhouette Score)
- 肘部法则 (Elbow Method)
- Calinski-Harabasz指数

### 2.2 主成分分析 (PCA)

PCA是最常用的降维算法。

**核心思想：**
- 找到数据方差最大的方向
- 将数据投影到这些方向上
- 保留主要信息，减少维度

**算法步骤：**
1. 数据标准化
2. 计算协方差矩阵
3. 计算特征值和特征向量
4. 选择最大的k个特征向量
5. 将数据投影到新的空间

### 2.3 层次聚类

层次聚类分为凝聚式和分裂式。

**凝聚式层次聚类：**
1. 将每个数据点作为一个簇
2. 合并距离最近的两个簇
3. 重复直到达到指定的簇数量

**距离度量：**
- 单链接：两个簇中最近点的距离
- 完全链接：两个簇中最远点的距离
- 平均链接：两个簇中所有点对的平均距离

## 三、模型评估与选择

### 3.1 评估指标

**分类指标：**
- 准确率 (Accuracy) = (TP+TN) / (TP+TN+FP+FN)
- 精确率 (Precision) = TP / (TP+FP)
- 召回率 (Recall) = TP / (TP+FN)
- F1分数 = 2 * Precision * Recall / (Precision + Recall)
- AUC-ROC：ROC曲线下面积

**回归指标：**
- 均方误差 (MSE) = (1/n) Σ(yᵢ - ŷᵢ)²
- 均方根误差 (RMSE) = √MSE
- 平均绝对误差 (MAE) = (1/n) Σ|yᵢ - ŷᵢ|
- R²分数 = 1 - (Σ(yᵢ - ŷᵢ)² / Σ(yᵢ - ȳ)²)

### 3.2 交叉验证

**K折交叉验证：**
1. 将数据分成K份
2. 每次用K-1份训练，1份验证
3. 重复K次，取平均性能

**留一法交叉验证：**
- K折的特殊情况，K等于样本数量
- 计算成本高，但结果更可靠

### 3.3 超参数调优

**网格搜索 (Grid Search)：**
- 遍历所有参数组合
- 计算成本高，但全面

**随机搜索 (Random Search)：**
- 随机采样参数组合
- 更高效，适合高维参数空间

**贝叶斯优化：**
- 基于历史结果选择下一组参数
- 更智能，但实现复杂

## 四、特征工程

### 4.1 特征缩放

**标准化 (Z-score)：**
z = (x - μ) / σ

**归一化 (Min-Max)：**
x' = (x - min) / (max - min)

### 4.2 特征编码

**独热编码 (One-Hot Encoding)：**
- 将类别变量转换为二进制向量
- 适合无序类别变量

**标签编码 (Label Encoding)：**
- 将类别变量转换为整数
- 适合有序类别变量

### 4.3 特征选择

**过滤法：**
- 基于统计指标选择特征
- 如相关系数、卡方检验、互信息

**包装法：**
- 使用模型评估特征子集
- 如递归特征消除 (RFE)

**嵌入法：**
- 在模型训练过程中选择特征
- 如Lasso回归、决策树特征重要性

## 五、深度学习基础

### 5.1 神经网络基础

**基本结构：**
- 输入层：接收原始数据
- 隐藏层：提取特征
- 输出层：产生预测

**激活函数：**
- Sigmoid：σ(x) = 1 / (1 + e^(-x))
- Tanh：tanh(x) = (e^x - e^(-x)) / (e^x + e^(-x))
- ReLU：f(x) = max(0, x)
- Leaky ReLU：f(x) = max(0.01x, x)

### 5.2 反向传播算法

**核心思想：**
- 计算损失函数对每个参数的梯度
- 使用梯度下降更新参数

**链式法则：**
∂L/∂w = ∂L/∂y * ∂y/∂w

### 5.3 优化算法

**梯度下降变体：**
- 批量梯度下降 (BGD)
- 随机梯度下降 (SGD)
- 小批量梯度下降 (Mini-batch GD)

**自适应学习率算法：**
- AdaGrad
- RMSProp
- Adam

### 5.4 常见架构

**卷积神经网络 (CNN)：**
- 卷积层：提取局部特征
- 池化层：降低维度
- 全连接层：分类

**循环神经网络 (RNN)：**
- 处理序列数据
- 存在梯度消失问题

**长短期记忆网络 (LSTM)：**
- 解决RNN的长期依赖问题
- 包含输入门、遗忘门、输出门

**Transformer：**
- 基于自注意力机制
- 并行计算效率高
- 用于NLP任务（BERT、GPT）

## 六、实践建议

### 6.1 数据预处理

1. 处理缺失值
2. 处理异常值
3. 特征缩放
4. 特征编码

### 6.2 模型选择流程

1. 从简单模型开始
2. 逐步增加复杂度
3. 使用交叉验证评估
4. 选择最佳模型

### 6.3 常见问题解决

**过拟合：**
- 增加数据量
- 使用正则化
- 减少模型复杂度
- 使用Dropout（深度学习）

**欠拟合：**
- 增加模型复杂度
- 添加更多特征
- 减少正则化强度
- 训练更长时间

**数据不平衡：**
- 过采样少数类
- 欠采样多数类
- 使用SMOTE
- 调整类别权重
