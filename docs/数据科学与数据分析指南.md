# 数据科学与数据分析指南

## 一、数据分析流程

### 1.1 CRISP-DM方法论

跨行业数据挖掘标准流程：
1. 业务理解：明确分析目标
2. 数据理解：收集和探索数据
3. 数据准备：清洗、转换、特征工程
4. 建模：选择和训练模型
5. 评估：验证模型效果
6. 部署：应用到实际业务

### 1.2 数据清洗

处理缺失值：删除、填充(均值/中位数/众数)、插值
处理异常值：IQR方法、Z-score方法
数据类型转换：astype()
重复值处理：drop_duplicates()

## 二、统计学基础

### 2.1 描述统计

集中趋势：均值、中位数、众数
离散程度：方差、标准差、极差、四分位距
分布形状：偏度(skewness)、峰度(kurtosis)

### 2.2 概率分布

正态分布(高斯分布)：钟形曲线
二项分布：离散型，多次伯努利试验
泊松分布：稀有事件计数
指数分布：事件间隔时间

### 2.3 假设检验

步骤：
1. 建立原假设H0和备择假设H1
2. 选择显著性水平alpha(通常0.05)
3. 计算检验统计量
4. 计算p值
5. 做出决策

常见检验：
- t检验：比较均值
- 卡方检验：分类变量独立性
- ANOVA：多组均值比较

### 2.4 回归分析

线性回归：y = b0 + b1*x + epsilon
多元回归：多个自变量
逻辑回归：分类问题
R-squared衡量模型拟合度

## 三、数据可视化

### 3.1 Matplotlib

基础绑图库，高度可定制。
折线图、柱状图、散点图、直方图、饼图、箱线图。

### 3.2 Seaborn

基于Matplotlib的高级可视化库。
统计图表：分布图、关系图、分类图。
美观的默认样式。

### 3.3 Plotly

交互式可视化库。
支持Web应用，可缩放、悬停显示详情。

### 3.4 可视化原则

选择合适的图表类型
标注轴标签和标题
颜色使用有意义
避免图表垃圾(chart junk)

## 四、特征工程

### 4.1 特征缩放

标准化(Z-score)：z = (x - mu) / sigma
归一化(Min-Max)：x' = (x - min) / (max - min)

### 4.2 特征编码

独热编码(One-Hot)：类别变量转二进制向量
标签编码(Label)：类别变量转整数
目标编码(Target)：用目标变量均值编码

### 4.3 特征选择

过滤法：相关系数、卡方检验、互信息
包装法：递归特征消除(RFE)
嵌入法：Lasso回归、决策树特征重要性

### 4.4 降维

PCA主成分分析：线性降维
t-SNE：非线性降维，适合可视化
LDA：有监督降维

## 五、A/B测试

### 5.1 流程

1. 提出假设
2. 确定指标
3. 计算样本量
4. 随机分组
5. 实施实验
6. 统计分析
7. 做出决策

### 5.2 注意事项

确保样本量充足
避免偷看结果(peeking)
考虑网络效应和溢出效应
多重比较校正

## 六、时间序列分析

### 6.1 基本概念

趋势(Trend)：长期变化方向
季节性(Seasonality)：周期性变化
噪声(Noise)：随机波动

### 6.2 分解方法

加法模型：Y = T + S + N
乘法模型：Y = T * S * N

### 6.3 预测方法

移动平均
指数平滑
ARIMA模型
Prophet(Facebook)
LSTM神经网络

## 七、机器学习基础

### 7.1 监督学习

分类：逻辑回归、决策树、随机森林、SVM、KNN
回归：线性回归、Ridge、Lasso、随机森林回归

### 7.2 无监督学习

聚类：K-means、DBSCAN、层次聚类
降维：PCA、t-SNE

### 7.3 模型评估

分类：准确率、精确率、召回率、F1、AUC-ROC
回归：MSE、RMSE、MAE、R-squared
交叉验证：K折交叉验证
