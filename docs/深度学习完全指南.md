# 深度学习完全指南

## 一、神经网络基础

### 1.1 什么是神经网络

神经网络是一种模仿生物神经系统的计算模型，由大量的人工神经元相互连接构成。每个神经元接收输入信号，经过加权求和和激活函数处理后产生输出。

### 1.2 感知机

感知机是最简单的神经网络，只有一个神经元。

数学模型：y = f(w1*x1 + w2*x2 + ... + wn*xn + b)

局限性：感知机只能解决线性可分问题，无法解决异或(XOR)问题。

### 1.3 多层感知机(MLP)

MLP由输入层、隐藏层和输出层组成，可以解决非线性问题。

前向传播过程：
- 输入层：接收原始数据
- 隐藏层：z = W*x + b, a = sigma(z)
- 输出层：产生预测结果

反向传播算法通过链式法则计算损失函数对每个参数的梯度。

梯度下降更新规则：theta = theta - alpha * dL/dtheta

### 1.4 激活函数

Sigmoid函数：sigma(x) = 1 / (1 + exp(-x))，输出范围(0,1)，存在梯度消失问题。

Tanh函数：tanh(x)，输出范围(-1,1)，零中心化。

ReLU函数：f(x) = max(0, x)，计算简单，缓解梯度消失，但存在死亡ReLU问题。

Leaky ReLU：f(x) = max(0.01x, x)，解决死亡ReLU问题。

GELU函数：GELU(x) = x * Phi(x)，Transformer中常用。

### 1.5 损失函数

回归损失：
- 均方误差(MSE)：L = (1/n) * sum(yi - yi_hat)^2
- 平均绝对误差(MAE)：L = (1/n) * sum|yi - yi_hat|
- Huber损失：结合MSE和MAE的优点

分类损失：
- 交叉熵损失：L = -sum(yi * log(yi_hat))
- 二元交叉熵：L = -[y*log(y_hat) + (1-y)*log(1-y_hat)]

### 1.6 优化算法

SGD(随机梯度下降)：theta = theta - alpha * grad(L)
带动量的SGD：v = beta*v - alpha*grad(L), theta = theta + v
AdaGrad：自适应学习率，对稀疏特征友好
RMSProp：指数衰减平均的AdaGrad变体
Adam：结合动量和RMSProp，最常用的优化器

## 二、卷积神经网络(CNN)

### 2.1 卷积操作

CNN通过卷积核在输入数据上滑动来提取局部特征。

输出尺寸计算：output = (input - kernel + 2*padding) / stride + 1

### 2.2 池化操作

最大池化：取局部区域的最大值
平均池化：取局部区域的平均值
全局平均池化：对整个特征图取平均

### 2.3 经典CNN架构

LeNet-5 (1998)：第一个成功的CNN，用于手写数字识别。

AlexNet (2012)：ImageNet竞赛冠军，使用ReLU和Dropout。

VGGNet (2014)：使用3x3小卷积核，网络深度16-19层。

GoogLeNet (2014)：引入Inception模块，使用1x1卷积降维。

ResNet (2015)：引入残差连接y = F(x) + x，解决梯度消失，可达152层。

EfficientNet (2019)：复合缩放方法，平衡深度、宽度和分辨率。

### 2.4 CNN应用

图像分类：迁移学习、数据增强
目标检测：YOLO系列、Faster R-CNN、SSD
语义分割：U-Net、FCN、DeepLab
图像生成：GAN、扩散模型

## 三、循环神经网络(RNN)

### 3.1 基础RNN

RNN专门处理序列数据，具有记忆功能。

隐藏状态：h_t = tanh(W_hh * h_{t-1} + W_xh * x_t + b)

问题：梯度消失/爆炸，难以捕捉长期依赖。

### 3.2 LSTM

LSTM通过门控机制解决长期依赖问题。

遗忘门：f_t = sigma(W_f * [h_{t-1}, x_t] + b_f)
输入门：i_t = sigma(W_i * [h_{t-1}, x_t] + b_i)
输出门：o_t = sigma(W_o * [h_{t-1}, x_t] + b_o)
细胞状态：C_t = f_t * C_{t-1} + i_t * tanh(...)
隐藏状态：h_t = o_t * tanh(C_t)

### 3.3 GRU

GRU是LSTM的简化版本，参数更少，训练更快。

更新门z_t和重置门r_t控制信息流动。

### 3.4 RNN应用

自然语言处理：语言模型、文本生成、机器翻译
语音识别：声学模型
时间序列预测：股票预测、天气预测

## 四、Transformer架构

### 4.1 注意力机制

Attention(Q, K, V) = softmax(Q*K^T / sqrt(d_k)) * V

### 4.2 多头注意力

MultiHead(Q, K, V) = Concat(head1, ..., headh) * W^O

### 4.3 位置编码

正弦位置编码：PE(pos, 2i) = sin(pos / 10000^(2i/d_model))

### 4.4 Transformer结构

编码器：多头自注意力 -> 残差连接+层归一化 -> FFN -> 残差连接+层归一化
解码器：带掩码的自注意力 -> 编码器-解码器注意力 -> FFN

### 4.5 BERT

双向Transformer编码器，预训练任务：掩码语言模型(MLM)和下一句预测(NSP)。
应用：文本分类、命名实体识别、问答系统。

### 4.6 GPT系列

自回归Transformer解码器。
GPT-1 (2018)：1.17亿参数
GPT-2 (2019)：15亿参数
GPT-3 (2020)：1750亿参数
GPT-4 (2023)：多模态能力

### 4.7 其他模型

T5：统一的文本到文本框架
LLaMA：Meta开源的大语言模型
DeepSeek：高效的大语言模型

## 五、生成对抗网络(GAN)

### 5.1 GAN基本原理

生成器G：从噪声z生成逼真数据
判别器D：区分真实数据和生成数据
训练目标：min_G max_D E[logD(x)] + E[log(1-D(G(z)))]

### 5.2 GAN变体

DCGAN：使用卷积层，稳定训练
WGAN：Wasserstein距离，梯度惩罚
StyleGAN：风格注入，高质量人脸生成
CycleGAN：无配对数据的图像翻译

### 5.3 GAN应用

图像生成、图像编辑、数据增强、文本到图像

## 六、正则化技术

L1正则化：产生稀疏解

Dropout：随机丢弃神经元
批归一化(BN)：加速训练，允许更大学习率
层归一化(LN)：常用于Transformer
权重初始化：Xavier、He初始化
